# 简介
1. 访问http://php.net/'，爬取该页内容
2. 使用BeautifulSoup清洗掉html标签
3. 分割出单词
4. 去掉停用词
5. 画出词频曲线
# 依赖
1. python 3.6.2
2. matplotlib
3. BeautifulSoup
4. nltk
# 安装virtualenv
```
pip3 install virtualenv
```
# 克隆项目
```
git clone git@github.com:NothinkingGao/nltk-record.git
```

# 运行
```
cd 
source venv/bin/activate
python nltk-lession1.py
```
